LED Dimmer - Arduino TutorialQuick Arduino project using the Elegoo Most Complete Starter Kit.
Wiring:Potentiometer Middle Pin -> Analog Pin A0Potentiometer Outer Pins -> 5V and GNDLED Long Leg (Anode) -> Digital Pin 9 (PWM)LED Short Leg (Cathode) -> 220Ω Resistor -> GNDHow to use: Upload the provided code to your Arduino.
